它們的「合理價」和「便宜價」是多少?  

金控和期貨、證券公司108年的業績都已經在1月10前公布了，業績亮晶晶。

      EPS出來了，只要知道「配息率」，就可算出未來的股利金額，也可推估多少錢是合理價、便宜價。      

假設某公司EPS = 3元，配息率70%，則

合理價 = EPS  X 配息率 X 20倍 = 3元 X 70% X 20 = 42元

     便宜價 = EPS  X 配息率 X 16倍 = 3元 X 70% X 16 = 33.6元

(「配息率」有些複雜，後面會談到)

 

      但其他公司的年報資料，都要到3月底才發佈，實在太慢了。       因此，上市櫃公司第3季季報資料出來以後，如何以這個做基礎，推估全年的EPS ，就是重要的工作。  

年度盈餘的概估

(後面有舉例:達欣工、福興、遠東銀 )      

全年度的業績，等於前3季業績加上「推估的」第4季的業績。

推估第4季的業績的方法有2種:      

第一， 以今年過去9個月的獲利水準，套用在第4季。優點是計算簡單，將錢9個月的業績除以9，乘以12。缺點是不考慮季節性。      

第二， 以去年第4季的獲利水準，套用在今年第4季，也就是加總最近4季的業績即可，缺點是要用到前一年的第4季資料，資料搜尋較不方便，而且每年大環境未必一樣。

資料搜尋方面，因為電腦方便，這一缺點是可以克服的。這方法另一優點是兼顧到某些在第四季特別興旺，與前三季淡季明顯不同的公司。            

全年業績估計公式一 = (今年前3季業績) / 9月 x  12 月            

全年業績估計公式二 = 最近4季業績(今年前3季業績 + 去年第4季業績)      

公式一、二的差別在於對第4季業績估計方法不同，兩種方法各有優缺點。        

(註:本文的舉例是採用公式二的估計方法。)

不確定因素

所謂「不確定因素」，就是將來出現的某些數值和我們想的不一樣，以致於我們得到的估計值偏高或偏低，會影響估計值以及投資的判斷，當然是很重要的。  

「不確定因素」主要有這兩個:

 配息率變化

 第4季業績變化

配息率的不確定性

公司的股利政策有一定的規則可循嗎?

 有些是有的，這又可分為按「固定金額」配發，以及按「固定比率」配發。

 有些則是沒有規則可循。

  第4季業績的不確定性

第4季的盈餘，和前3季比較，可能有下面幾種情形:  

第一，公司業務性質具有一致性的盈餘，用前9月的盈餘推估第4季的每股盈餘，很合適。  

第二，公司業務性質第4季是旺季，如果用前9月的盈餘水準推估，結果會偏低。  

第三，公司第4季是淡季，或是在年底宣布資產減損，認列虧損，如果用前9月的盈餘水準推估，結果會偏高。  

第四，公司第4季遇到意外衝擊，若用前9月的盈餘水準推估，結果會偏高許多。

【圖表】推估第4季盈餘的四種狀況

種類 舉例
推估結果很合適

 公司全年收入及盈餘具有一致性的特性，

如用前9月的盈餘水準推估全年盈餘，結果很合適。例如大台北(9908)、高鐵(2633)、中保(9917)。
推估結果偏低

 公司第4季是旺季，用前9月的盈餘水準推估全年盈餘，結果偏低。

例如電子業為了12月以後的銷售旺季而接受訂單，所以旺季發生在9-11月。

 又例如工程公司、建設公司趕在第4季完工結帳，業績會較前3季好。 推估結果偏高。

 公司第4季是淡季，或是在年底宣布資產減損，認列虧損， 如果用前9月的盈餘水準推估，結果就可能偏高。

 例如欣陸、光寶、宏碁都曾在年底宣布資產減損。
推估結果偏高甚多  107年第四季股價下跌、美國利率上升， 導致擁有眾多金融資產(股票及債券)的公司，受到甚大的衝擊，如用前9月的盈餘水準推估全年盈餘，結果會偏高甚多。  例如壽險公司及壽險金控公司。

溫老師研究的叮嚀

      本文是從各公司前3季的業績，討論如何推估合理股價的方法，很實用，

但也因為是推估，其中就含有一些不確定性，

為了克服這些不確定性， 一定要多研究它的基本面，

並多觀察利率、匯率、股市變化等。  

      例如達欣工(2535)是一家工程公司，工程公司的特性是收入要看工程進度是否達到合約收款的程度， 所以各季的收入不均勻，因此，以去年同期業績套用到今年第4季的業績，還是有精準性的顧慮。  

      至於福興(9924)第3季業績亮麗，但108年第4季新台幣匯率又有較大升值， 需考慮該公司為外銷比重很高的公司，盈餘往往受新台幣匯率波動影響，所以還是要繼續觀察。  

     在遠東銀(2845)方面，前3季盈餘成長10.76%，且公司性質在第4季的變數也不至於太大， 推估的結果是合理價比市價高一些，似乎是仍有機會上漲的空間，值得留意。
